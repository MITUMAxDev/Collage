Debugging 
Lua มีไลบรารีสำหรับการดีบักที่รวมฟังก์ชันพื้นฐานทั้งหมดที่ช่วยให้เราสามารถสร้างดีบักเกอร์ของเราเองได้
แม้ว่า Lua จะไม่มีดีบักเกอร์ที่ติดตั้งมาในตัว แต่ก็มีดีบักเกอร์มากมายสำหรับ Lua ที่สร้างขึ้นโดยนักพัฒนาหลายคน ซึ่งส่วนใหญ่เป็นโอเพนซอร์ส

ฟังก์ชันที่มีอยู่ในไลบรารีการดีบัก Lua แสดงต่อไปนี้พร้อมกับการใช้งาน
        
    1) debug()
       เข้าสู่โหมดโต้ตอบสำหรับการดีบัก ซึ่งจะยังคงทำงานอยู่จนกว่าคุณจะพิมพ์ 'continue' 
       แล้วกด Enter คุณสามารถตรวจสอบตัวแปรในโหมดนี้โดยใช้ฟังก์ชันอื่น ๆ
    
    2) getfenv(object)
       ส่งคืนสภาพแวดล้อมของวัตถุ (object)
    
    3) gethook(optional thread)
       ส่งคืนการตั้งค่าตะขอปัจจุบันของเธรดเป็นค่าสามค่า: ฟังก์ชันตะขอปัจจุบัน, 
       หน้ากากตะขอปัจจุบัน, และจำนวนตะขอปัจจุบัน
    
    4) getinfo(optional thread, function or stack level, optional flag)
       ส่งคืนตารางที่มีข้อมูลเกี่ยวกับฟังก์ชัน คุณสามารถให้ฟังก์ชันโดยตรงหรือ 
       ใช้ตัวเลขเป็นค่าของฟังก์ชัน ซึ่งหมายถึงฟังก์ชันที่ทำงานที่ระดับของ stack ในเธรด 
       ระดับ 0 คือฟังก์ชันปัจจุบัน (getinfo เอง) ระดับ 1 คือฟังก์ชันที่เรียก getinfo 
       และอื่น ๆ หากฟังก์ชันเป็นตัวเลขที่มากกว่าจำนวนฟังก์ชันที่ใช้งานอยู่ 
       getinfo จะส่งกลับค่า nil
    
    5) getlocal(optional thread, stack level, local index)
       ส่งคืนชื่อและค่าของตัวแปรโลคัลด้วยดัชนีโลคัลในฟังก์ชันที่ระดับ stack 
       ส่งคืนค่า nil หากไม่มีตัวแปรโลคัลที่มีดัชนีที่กำหนด และทำให้เกิดข้อผิดพลาดเมื่อ 
       เรียกด้วยระดับที่อยู่นอกช่วง
    
    6) getmetatable(value)
       ส่งคืน metatable ของวัตถุที่กำหนดหรือศูนย์หากไม่มี metatable
    
    7) getregistry()
       ส่งคืนตารางรีจิสทรี ซึ่งเป็นตารางที่กำหนดไว้ล่วงหน้า สามารถใช้โดยรหัส C 
       ใดก็ได้เพื่อเก็บค่า Lua ที่ต้องการจัดเก็บ
    
    8) getupvalue(function, upvalue index)
       ส่งคืนชื่อและค่าของค่า upvalue ด้วยดัชนีที่กำหนดในฟังก์ชัน 
       ฟังก์ชันจะคืนค่าศูนย์หากไม่มีค่า upvalue ที่กำหนด
    
    9) setfenv(function or thread or userdata, environment table)
       ตั้งค่าสภาพแวดล้อมของวัตถุที่กำหนดให้กับตารางที่กำหนด ส่งคืนวัตถุ
    
    10) sethook(optional thread, hook function, hook mask string with "c" and/or "r" and/or "l", optional instruction count)
        ตั้งค่าฟังก์ชันที่กำหนดเป็นตะขอ มาสก์สตริงและการนับจำนวนอธิบายว่าเมื่อใด 
        จะเรียกเบ็ด ที่นี่ c, r และ l ถูกเรียกทุกครั้งที่ Lua เรียก, ส่งกลับ, 
        และป้อนรหัสทุกบรรทัดในฟังก์ชันตามลำดับ
    
    11) setlocal(optional thread, stack level, local index, value)
        กำหนดค่าให้กับตัวแปรโลคัลด้วยดัชนีโลคัลในฟังก์ชันที่ระดับ stack 
        ส่งคืนค่า nil หากไม่มีตัวแปรโลคัลที่มีดัชนีที่กำหนด 
        และทำให้เกิดข้อผิดพลาดเมื่อเรียกด้วยระดับที่อยู่นอกช่วง
    
    12) setmetatable(value, metatable)
        ตั้งค่า metatable สำหรับวัตถุที่กำหนดให้กับตารางที่กำหนด (ซึ่งอาจเป็นศูนย์)
    
    13) setupvalue(function, upvalue index, value)
        กำหนดค่าให้กับค่า upvalue ด้วยดัชนีที่กำหนดในฟังก์ชัน 
        ฟังก์ชันจะคืนค่าศูนย์หากไม่มีค่า upvalue ที่กำหนด
    
    14) traceback(optional thread, optional message string, optional level argument)
        สร้างข้อความแสดงข้อผิดพลาดเพิ่มเติมพร้อมการย้อนกลับ
